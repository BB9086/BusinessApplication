**About this app:** This is a sample business application (Model-View-ViewModel (MVVM) design pattern) used by owners of one or more restaurants, which allows them to track their daily, monthly and annual income. MetroLog library is used to enable client-side error logging and App Center is also integrated. HttpClient class in mobile app is used to consume ASP.NET Web API in order to get data that is shown to the restaurant owners. ASP.NET Core Web Api is deployed to Azure App Service.
